<H1>
여러 커뮤니티들의 핫딜, 특가 게시판의 게시판을 한 곳에서 모아 볼 수 있는 사이트
</H1>

ㅁ 개발 환경
  - SpringBoot 2.*
  - Maven
  - java 8
  - vue 2.5
  - thymeleaf 
  
  
ㅁ 설명
  - 너굴 핫딜 모아모아는 루리웹, 뽐뿌 등의 여러 핫딜, 특가 게시판의 정보를 기반으로
  사용자들이 한 화면에서 모든 게시판의 글을 확인 할 수 있도록 
  구현되어 있습니다.
  
  
ㅁ 추후 구현 예정 기능
  - 키워드 설정 및 이메일 알림기능 
    -> 알림 기능을 통해 키워드와 이메일을 설정하면 해당 키워드가 포함 된 글이 오게 되면 이메일로 알림 서비스를 해주는 기능
